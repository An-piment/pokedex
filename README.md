## Project Pokedex 

## Informações

Projeto de um site em que é possível procurar um pokemon de qualquer geração e verificar seus status dentro do jogo.

É utilizado uma API em que é possível receber todos as informações de cada pokemon.

## Instalação

Para instalar as dependências do projeto utilize o comando:

```sh
$ npm install
```

Em seguida, para abrir o projeto, utilize o comando:

```sh
$ npm run dev
```
